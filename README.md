#### Сборка
```
mvn clean install
```
>Java SDK 18 / Maven 3.9.9

#### Запуск
```
java -jar utilityFilter-1.0.jar [-s | -f] [-a] [-p <prefix>] [-o <out_path>] [<path_in>.txt]
```

#### Параметры
- **[-p \<prefix\>]**: флаг указывающий префикс исходящего файла. Необходимо указывать префикс после флага.
- **[-o <out_path>]**: флаг указывающий путь до исходящих файлов. Создаются новые, если по указанному пути нет существующих файлов. Необходимо указывать путь после флага.
- **[<path_in>.txt]**: указывает путь до входящий файлов. При необходимости можно указывать несколько. Читает только файлы с расширением .txt
- **[-s | -f]**: **[-s]** вывод краткой статистики по записанным в файл элементам, **[-f]** вывод полной статистики.
- **[-a]**: флаг записи в конец в исходящего файла.
  
>Параметры указываются в произвольном порядке. При указании нескольких флагов с параметрами, при запуске будет учитываться только последний(относится к флагам **[-p \<prefix\>]** и **[-o <out_path>]**)
